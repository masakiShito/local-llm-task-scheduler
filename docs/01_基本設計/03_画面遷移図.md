# 基本設計書：画面一覧（MVP）

## 1. 本書の目的

本書は、MVP で提供する画面を一覧化し、画面の責務・主要機能・入出力（参照/更新対象）を明確にする。
画面遷移図、API 一覧、詳細設計、実装の前提資料とする。

---

## 2. 画面 ID 命名ルール

- 画面 ID は `SCR-XXX` 形式とする
- 大分類ごとに番号を割り当てる
  - 100 番台：ホーム/ダッシュボード
  - 200 番台：タスク
  - 300 番台：固定予定
  - 400 番台：計画（Plan）

---

## 4. MVP の割り切り（画面機能）

- 計画（Plan）の「手動編集（ドラッグ&ドロップ）」は MVP 必須としない
- 計画ブロックの「ロック」と「ロック維持再生成」は Phase1.5 以降で対応予定
- ログイン/ユーザー管理画面は MVP 対象外（単一ユーザー想定）
