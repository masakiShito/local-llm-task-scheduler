# 詳細設計：エラーレスポンス仕様（MVP）

## 1. 本書の目的

API が返却するエラーレスポンスの構造と運用ルールを定義する。

---

## 2. 共通フォーマット

    error:
      message_id: E-0400
      message: 入力内容を確認してください
      field_errors: []

---

## 3. field_errors の詳細

### 3.1 用途

- 入力項目単位でのエラー表現

### 3.2 表現ルール

- field：リクエスト JSON キー
- message_id：原則 E-0400
- message：既定文言（UI で差し替え可）

---

## 4. 404 エラー

    error:
      message_id: E-0404
      message: 対象データが存在しません
      field_errors: []

---

## 5. 500 エラー

    error:
      message_id: E-0500
      message: サーバでエラーが発生しました
      field_errors: []

---

## 6. UI 表示ルール（参考）

- field_errors：項目直下に表示
- message：画面上部に表示
